# راهنمای نصب PassWall2 روی OpenWrt 25.x

## 🌐 Language / زبان

- 🇮🇷 فارسی (صفحه فعلی)
- 🇬🇧 [English](README.md)
این مخزن با هدف ارائه یک راهنمای عملی برای نصب و راه‌اندازی PassWall2 روی نسخه‌های جدید OpenWrt ایجاد شده است.

بسیاری از راهنماهای موجود مربوط به نسخه‌های قدیمی OpenWrt هستند و کاربران هنگام نصب PassWall2 روی نسخه‌های جدید با مشکلاتی مانند وابستگی‌ها (Dependencies)، ناسازگاری پکیج‌ها و مشکلات Xray مواجه می‌شوند.

این راهنما بر اساس یک نصب واقعی و موفق تهیه شده است.

# بهینه‌سازی فضای ذخیره‌سازی

یکی از محدودیت‌های Google WiFi فضای ذخیره‌سازی داخلی است.

برای جلوگیری از پر شدن حافظه فلش، Xray در مسیر زیر اجرا شده است:

```text
/tmp/xray
```

به جای:

```text
/usr/bin/xray
```

مسیرهای فعال در سیستم تست شده:

```text
Xray App Path:
/tmp/xray

Sing-Box App Path:
/usr/bin/sing-box

Hysteria App Path:
/usr/bin/hysteria

Geoview App Path:
/usr/bin/geoview
```

این روش باعث کاهش مصرف حافظه داخلی روتر می‌شود.
